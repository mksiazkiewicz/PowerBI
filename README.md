# English

## About

Welcome to my Power BI portfolio.

This repository contains a collection of Business Intelligence projects created using **Microsoft Power BI**. Each project focuses on solving a different analytical problem and demonstrates the complete BI workflow—from data acquisition and preparation to data modeling, DAX calculations, and interactive dashboard development.

Some projects use local datasets, while others retrieve data directly from external sources such as REST APIs or Microsoft SQL Server.

---

## Projects

### Adventure Works

Business Intelligence dashboard built using the Adventure Works database.

The project demonstrates a real-world BI workflow. Data was first prepared in **Microsoft SQL Server**, where I created SQL views containing the required business information. Power BI was then connected directly to these views, allowing efficient data modeling and dashboard creation.

**Technologies**

* Microsoft SQL Server
* SQL Views
* Power BI
* Power Query
* DAX

---

### Weather Data Analysis

Interactive dashboard based on current weather data provided by the **Institute of Meteorology and Water Management (IMGW)**.

Data is retrieved directly from the public REST API in JSON format using Power Query (`Web.Contents()` and `Json.Document()`), transformed, and visualized in Power BI.

**Technologies**

* Power BI
* Power Query (M)
* REST API
* JSON

---

### Electric Cars Analysis

Dashboard presenting analysis of electric vehicle data.

The project explores market trends and vehicle characteristics using interactive visualizations.

**Technologies**

* Power BI
* Power Query
* CSV

---

### Sales Report

Interactive sales dashboard designed to monitor key business indicators.

The report includes revenue analysis, product performance, store analysis, and KPI monitoring.

**Technologies**

* Power BI
* DAX
* Power Query
* Excel
* CSV

# Polski

## O repozytorium

Repozytorium zawiera moje projekty wykonane w **Microsoft Power BI**.

Każdy projekt przedstawia inny problem biznesowy lub analityczny i pokazuje pełny proces tworzenia rozwiązania Business Intelligence – od pozyskania danych, przez ich przygotowanie i modelowanie, aż po stworzenie interaktywnego dashboardu.

W projektach wykorzystuję zarówno lokalne źródła danych, jak i połączenia z **Microsoft SQL Server** oraz publicznymi interfejsami **REST API**.

---

## Projekty

### Adventure Works

Dashboard analityczny oparty na bazie Adventure Works.

W projekcie przygotowałam własne widoki w **Microsoft SQL Server**, zawierające odpowiednio przetworzone dane biznesowe. Następnie Power BI został połączony bezpośrednio z utworzonymi widokami SQL, co odzwierciedla sposób pracy stosowany w rzeczywistych projektach Business Intelligence.

**Technologie**

* Microsoft SQL Server
* SQL
* Widoki SQL
* Power BI
* Power Query
* DAX

---

### Analiza danych pogodowych

Interaktywny raport wykorzystujący aktualne dane pogodowe udostępniane przez **IMGW**.

Dane pobierane są bezpośrednio z publicznego API w formacie JSON przy użyciu funkcji `Web.Contents()` oraz `Json.Document()` w Power Query.

**Technologie**

* Power BI
* Power Query (M)
* REST API
* JSON

---

### Analiza samochodów elektrycznych

Dashboard prezentujący analizę danych dotyczących samochodów elektrycznych oraz trendów rynkowych.

**Technologie**

* Power BI
* Power Query
* CSV

---

### Raport sprzedaży

Dashboard biznesowy przedstawiający analizę sprzedaży oraz najważniejsze wskaźniki KPI.

Raport umożliwia analizę przychodów, produktów, sklepów oraz wyników sprzedażowych.

**Technologie**

* Power BI
* DAX
* Power Query
* Excel
* CSV
