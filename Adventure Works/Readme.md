# Adventure Works - Sales Analytics Dashboard

## Overview

This project presents a business intelligence dashboard built using the Adventure Works database.

Instead of importing raw tables directly into Power BI, I first designed and created **SQL views in Microsoft SQL Server** to prepare and organize the required business data. Power BI was then connected directly to these views, creating a clean and efficient reporting layer.

This approach reflects a real-world BI workflow, where data transformation and preparation are performed within the database before visualization.

## Project Workflow

1. Analyzed the Adventure Works database structure.
2. Created SQL views containing the required business data.
3. Connected Power BI directly to the SQL Server views.
4. Built the data model and created DAX measures.
5. Designed an interactive dashboard with business KPIs and visualizations.
