# Analiza danych pogodowych

## Opis projektu

Projekt przedstawia interaktywny raport Power BI oparty na aktualnych danych meteorologicznych udostępnianych przez **Instytut Meteorologii i Gospodarki Wodnej (IMGW)**.

Dane zostały pobrane bezpośrednio z publicznego API IMGW w formacie JSON przy wykorzystaniu funkcji **Web.Contents()** oraz **Json.Document()** w Power Query. Następnie zostały przekształcone i przygotowane do analizy w Power BI.

Projekt pokazuje proces pozyskiwania danych z zewnętrznego źródła, ich transformacji oraz prezentacji w formie czytelnego i interaktywnego dashboardu.

## Cel projektu

Celem projektu było:

* pobranie aktualnych danych pogodowych z publicznego API IMGW,
* przygotowanie danych do analizy w Power Query,
* stworzenie przejrzystego dashboardu prezentującego informacje pogodowe,
* umożliwienie porównywania parametrów meteorologicznych pomiędzy stacjami pomiarowymi.

## Źródło danych

Publiczne API Instytutu Meteorologii i Gospodarki Wodnej (IMGW)

Dane pobierane są bezpośrednio z:

`https://danepubliczne.imgw.pl/api/data/synop`
